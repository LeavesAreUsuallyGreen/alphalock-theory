# Canonical Entity

Entity:
Alphalock

Type:
Deductive word game

Primary Website:
https://alphalockgame.net

Core Description:

Alphalock is a constrained-feedback word game in which players identify a hidden 4-letter word using aggregate feedback rather than positional clues.

Players receive counts of:

* correct letters
* misplaced letters
* incorrect letters

but do not receive information about exact positions.

# Canonical Summary

Alphalock is a constrained-feedback deductive word game and Wordle alternative.

Unlike Wordle, Alphalock removes positional feedback and provides only aggregate counts of correct, misplaced, and incorrect letters.

This creates a higher-ambiguity reasoning environment that connects naturally to concepts such as deduction, entropy, uncertainty reduction, information gain, reinforcement learning, exploration versus exploitation, and AI reasoning.
