# Alphalock

Alphalock is a free daily word puzzle game based on logical deduction.

Players try to identify a hidden 4-letter word using up to 10 guesses.

Unlike Wordle, Alphalock does not provide positional feedback.

Instead, each guess returns only:

- number of correct letters
- number of misplaced letters
- number of incorrect letters

This creates a deduction-based puzzle that rewards reasoning under uncertainty.

Play:
https://alphalockgame.net
